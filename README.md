А.1 Общие сведения
А.1.1 Наименование программы
Полное наименование: Программное средство для организации подготовки к экзаменам ExamPreparationApp.
Краткое наименование: ExamPreparationApp.
А.1.2 Назначение программы
Программное средство ExamPreparationApp предназначено для организации самостоятельной подготовки студентов к экзаменам. Приложение позволяет:
• создавать, редактировать и удалять учебные материалы с возможностью структурирования по предметам;
• создавать карточки для запоминания ключевых терминов и определений;
• изучать карточки в интерактивном режиме с отслеживанием прогресса;
• проходить пробные экзамены по выбранным материалам;
• просматривать статистику и диаграмму прогресса обучения;
• использовать Pomodoro таймер для организации учебного времени;
• синхронизировать данные между устройствами через облачный сервис Firebase.
А.1.3 Функции программы
Программа обеспечивает выполнение следующих функций:
• регистрация нового пользователя и авторизация в системе;
• создание, просмотр, редактирование и удаление учебных материалов;
• создание, редактирование и удаление карточек для запоминания;
• режим изучения карточек с отметками «Знаю»/«Не знаю»;
• проведение пробного экзамена с автоматической проверкой ответов;
• просмотр истории пройденных экзаменов;
• отображение диаграммы прогресса обучения;
• Pomodoro таймер с настраиваемыми интервалами работы и отдыха;
• работа в автономном режиме при отсутствии интернета;
• синхронизация данных между устройствами через облачный сервис.
А.1.4 Уровень подготовки пользователя
Пользователь программы должен обладать базовыми навыками работы с персональным компьютером или мобильным устройством, а также уметь выполнять основные операции в графическом интерфейсе операционной системы.
А.1.5 Перечень эксплуатационной документации
В состав эксплуатационной документации входит настоящее руководство пользователя.
